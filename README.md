# ff-homepage

Homepage-Entwurf für **Fördermittel-Finder** — ein gemeinsames Angebot von
Prospero Beratungsgesellschaft mbH und ES | Unternehmensberatung.

Dieses Repository enthält ausschließlich die öffentliche Marketing-Website
(statisches HTML/CSS, keine Kundendaten, keine Berichte). Gedacht für GitHub
Pages (Branch `main`, Root).

Quelle/Weiterentwicklung der eigentlichen Fördermittel-Recherche- und
Berichts-Tools liegt im privaten Repo `Foerdermittel`.

## Vorschau lokal starten

```bash
python3 -m http.server 8000
```

Dann `http://localhost:8000` öffnen.
